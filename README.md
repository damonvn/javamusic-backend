# javamusic-backend

## Project Overview

This is a demo project for a music management backend using Spring Boot.

## Technology Stack

-   **Java Version:** 17
-   **Spring Boot Version:** 3.3.2
-   **Build Tool:** Maven
-   **Database:** MongoDB

## Prerequisites

To build and run this project, you need to have the following installed:

-   Java 17 or higher
-   Maven 3.6 or higher
